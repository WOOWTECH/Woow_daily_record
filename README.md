# Woow Daily Record

A comprehensive family management application built with Next.js.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- Multi-site (household) management
- Health tracking
- Calendar management
- Notes and tasks
- Finance tracking
- Baby growth records

## Tech Stack

- Next.js 16
- TypeScript
- Supabase
- Zustand
- Tailwind CSS
